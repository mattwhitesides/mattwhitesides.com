Using fgets instead of fscanf

Fgets takes in a character buffer, a size limit and a file pointer.

Unlike fscanf which tries to determine the pattern of a file,

fgets puts in the character byte data read in from the file

and puts it raw into your char array.

fgets will read until your size limit is reached or gets to the end of a line.


FILE* filePtr = fopen(fileName, "r");

char buffer[100];


while (fgets(buffer, 100, filePtr) != NULL) {

//The variable buffer

//now holds all the file data
